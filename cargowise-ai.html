Available Today · CargoWise Value Pack

Three tools. One question: is your workflow ready for them?

Tool 1

Commercial Invoice Parsing

Reads commercial invoices and extracts line-item data into CargoWise. Eliminates manual data entry on the document processing step. Requires clean milestone configuration to trigger correctly.

Tool 2

ComplianceWise

Automated compliance screening against trade sanctions and restricted party lists. Flags shipments that need human review before customs lodgement. Only works if it fires at the right point in the workflow.

Tool 3

Classification Assistant

AI-assisted tariff classification. Suggests HS codes based on cargo description. Requires a human check at lodgement — statutory responsibility stays with the broker.

The Implementation Gap

Having the tools switched on is not the same as having them work.

The most common failure mode: firms activate the Value Pack AI features against a CargoWise configuration that was never designed for agents. The document parsing triggers at the wrong milestone. The compliance screening fires after the customs entry is already lodged. The classification suggestions go to an inbox nobody watches.

ViAGO audits your CargoWise configuration, maps your sea import workflow against the agent touchpoints, and configures the system so the tools actually do what they're supposed to.
